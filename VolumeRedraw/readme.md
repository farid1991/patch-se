# Volume Redraw
Redraw volume control GUI with custom skin.  
Use `VolumeSkinEditor.elf` to edit skin.

## How to use
- set `.vsf` extension openwith to `VolumeSkinEditor.elf` using `extedit,elf`

- **C510 R1HA035**
- **C903 R1GA028**
